# Textile Factory Website

A responsive website for a textile factory with product catalog, feedback form and admin panel.

## 🚀 Live Demo

**[➜ Open Website](https://rossiav802-code.github.io/textile-factory-website)**

## ✨ Features

- 📦 Product catalog with categories
- 📬 Feedback form with validation
- 🔐 Admin panel for content management
- 📱 Fully responsive design
- 🎨 Clean and modern UI
